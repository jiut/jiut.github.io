发表于 2022-03-09

原文链接

方式一：通过修改.gitignore文件实现

打开通过git clone命令获取的项目

打开.gitignore文件

配置语法：
以斜杠“/”开头表示目录；
以星号“*”通配多个字符；
以问号“?”通配单个字符
以方括号“[]”包含单个字符的匹配列表；
以叹号“!”表示不忽略(跟踪)匹配到的文件或目录；

方式二

通过命令实现
这种方式忽略单个文件

打开通过git clone命令获取的项目
输入命令
git update-index --assume-unchanged 文件名

取消忽略
git update-index --no-assume-unchanged 文件名
